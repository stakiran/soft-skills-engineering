# index maker
`python indexmaker.py` を実行すると、index.md を生成する

## index.md の中身
ヘッダ、ボディ、フッタから成る

```markdown
(header)

(body)

(footer)
```

ヘッダとフッタはハードコードで良い。

ボディはファイルを走査して索引をつくる。

### ヘッダ
以下の内容にする。

```
# Soft Skills Engineering
- [GitHub](https://github.com/stakiran/soft-skills-engineering)

## TOC
```

### フッタ
空で良い。

### ボディ
- カレントディレクトリにある `{Number}_{Name}.md` を走査する
    - Number は `00`, `01`, `13` など二桁の数字である
- 走査は昇順に行う
- なおファイル名として以下がありえる
    - 日本語版のファイル `{Number}_{Name}.md`
    - 英語版のファイル ``{Number}_{Name}_EN.md``
    - つまり英語版には `_EN` がついている
- ファイルのタイトルは、ファイル内容の先頭行を見る。ただし見出し記法は除外する
    - これは正しくない → `# 本書について`
    - これは正しい → `本書について`
    - つまり先頭の `# ` を削除すればタイトルになる

走査後、以下のフォーマットで仕上げる。

```
- (Number)
    - [日本語ファイル名のタイトル](日本語ファイル名)
    - [英語ファイル名のタイトル](英語ファイル名)
- (Number)
    - [日本語ファイル名のタイトル](日本語ファイル名)
    - [英語ファイル名のタイトル](英語ファイル名)
- ...
```

例を示す

```
- 00
    - [本書について](00_about.md)
    - [About This Book](00_about_EN.md)
- 01
    - ...
- ...
```
